¿Qué diferencias y semejanzas hay entre Java y JavaScript?
  • Semejanzas: las primeras cuatro letras. ambos soportan programacion orientada a objetos y tienen sintaxis basada en C.
  • Diferencias: JS es un lenguaje de scripting con tipos dinámicos y Java es un lenguaje estrictamente tipado y estático.
